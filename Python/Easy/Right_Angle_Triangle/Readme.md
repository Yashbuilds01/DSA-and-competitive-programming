# Consecutive Numbers Right-Angle Triangle Pattern

## 📝 Problem Statement
Given a positive integer `N`, your task is to print a right-angle triangle pattern of consecutive numbers of height `N`. 

This is a functional problem, so you do not need to handle input reading. You only need to complete the function `pattern()` which takes the integer `n` as a parameter.

## 📥 Constraints
- $1 \le N \le 100$

## 📤 Output Format
Print a right-angle triangle of numbers of height `N`, where each row contains consecutive integers starting from `1` up to the row number, separated by spaces.
